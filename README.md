# RL QR Generator Pro

A modern, gradient-styled QR code generator — built for business cards, wedding invitations, WiFi sharing, and anything else that needs a QR code that doesn't look like it came from a spammy 2009-era generator.

**🔗 Try it live:** [lakshan.vercel.app/qr](https://lakshan.vercel.app/qr)

100% client-side — nothing you type or upload ever leaves your browser. No ads, no tracking, no server.

---

## ✨ Features

### Content types
- **Link / URL** — for websites, invitation pages, menus, portfolios
- **Plain text** — any message, directions, notes
- **WiFi** — network name, password, and security type (WPA/WEP/none) — scan to auto-connect
- **Contact card (vCard)** — name, phone, email — scan to save a contact

### Design & styling
- **Gradient foreground** — 10 built-in color presets, or pick your own start/end colors
- **Gradient shape** — Linear or Radial
- **Gradient angle control** — full 0°–360° rotation slider for linear gradients
- **6 dot styles** — Rounded, Dots, Classy, Classy edge, Square, Extra-rounded
- **3 corner frame styles** — Rounded, Dot, Square
- **Custom background color**, or fully **transparent background** (for overlaying on flyers/designs)

### Logo embedding
- Upload any image/logo to sit in the center of the QR code
- Adjustable logo size (10%–40%)
- Automatically switches to high error-correction so the code still scans reliably with a logo covering part of it

### Pro finishing touches
- **Caption badges** — add a "Scan me", "Scan to join", "Scan for WiFi" pill under the code, or write your **own custom text** with a custom color
- **Export resolution control** — Standard (600px, web/social), High (1000px, flyers/cards), or Ultra (2000px, large print/banners)
- Live animated preview with a scan-line effect on every change

### Export
- Download as **PNG**, **SVG**, or **JPG**
- Exports render fresh at your chosen resolution — not just an upscaled screenshot, so print output stays crisp

### Built for real use
- Dark mode by default, clean green/mint gradient theme
- Fully responsive — works great on mobile
- Fast — debounced live preview, no lag while typing
- No sign-up, no server round-trip, no data collection
